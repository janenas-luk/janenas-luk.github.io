# Introduction

vuci implemented in vue.js (opens new window) and Ant Design of Vue (opens new window), inspired by LuCI2 (opens new window).

vuci uses json-rpc (opens new window) to communicate with OpenWrt subsystems.

# Features

  • Fast: Use static pages; access any kind of system data via ubus
  • Vue: Enjoy the dev experience of Vue
  • flexible: Flexible, easy to customize UI
