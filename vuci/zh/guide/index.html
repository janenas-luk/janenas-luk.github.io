# 介绍

vuci使用vue.js (opens new window)和Ant Design of Vue (opens new window)实现，灵感来自于LuCI2 (opens new window)。

vuci使用json-rpc (opens new window)和OpenWrt子系统通信。

# 特性

  • 快: 使用静态页面；通过ubus存取各种系统数据
  • Vue: 享受现代化的前端框架 Vue 所带来的开发体验
  • 灵活: 灵活的框架，方便自定义UI
